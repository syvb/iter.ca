Here’s a list of the computer stuff I use.

## Hardware
- One main laptop I use is [Framework Laptop 13](https://frame.work/laptop13) that runs NixOS; I’ve [open-sourced my Nix config](https://github.com/syvb/iter.ca/blob/master/conf/nix/base.nix) if you’re interested in seeing how I’ve configured it. 
- My other main laptop is a MacBook Pro running macOS. (I’ve tried Asahi Linux for some experiments too.)
- I use a Google Pixel 8 as main main phone. (I used to use a [PinePhone Pro](https://pine64.org/devices/pinephone_pro/) but I needed my phone to do a bunch of things mobile Linux doesn’t support.)

## Development
I use VSCodium (open source version of VS Code without the non-free blobs) to do most of my development work. I use the [rust-analyzer](https://rust-analyzer.github.io) extension when working with Rust; it provides a great IDE experience for navigating and editing Rust codebases.
