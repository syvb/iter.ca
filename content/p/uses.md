Here’s a list of the stuff I use in my life, based on [uses.tech](https://uses.tech/). See also [my post about software I use](https://iter.ca/post/programs-like/).

## Hardware
- One main laptop I use is [Framework Laptop 13](https://frame.work/laptop13) that runs NixOS; I’ve [open-sourced my Nix config](https://github.com/syvb/iter.ca/blob/master/conf/nix/base.nix) if you’re interested in seeing how I’ve configured it. 
- My other main laptop is a 2021 14-inch MacBook Pro running macOS. (I’ve tried Asahi Linux for some experiments too.)
- I use a Google Pixel 8 as my main phone. I used to use a [PinePhone Pro](https://pine64.org/devices/pinephone_pro/) but I needed my phone to do a bunch of things mobile Linux doesn’t support.
- I have a [Pixel Watch 4](https://store.google.com/product/pixel_watch_4?hl=en-US&pli=1); the main value I get from it is sleep tracking (it provides really useful metrics about my sleep process and duration). It also provides me with a bunch of other useful health stuff like heart-rate tracking and step counting.
- I use a paper-sized [Kindle Scribe](https://www.amazon.ca/dp/B0CZ9TBL1H) mostly for reading stuff (you can send PDFs/EPUBs to it in addition to things you buy on Amazon), although it’s also pretty handy for jotting down notes.
- I use my Aranet 4 Home for tracking CO<sub>2</sub> (and temperature/humidity) in rooms. It has a very good CO<sub>2</sub> detector, but is probably overkill for most people (it costs more than most other CO<sub>2</sub> detectors because the sensor it uses is expensive).
- I have an Nintendo Switch (OLED model) I do most of my video gaming on

## Clothing
Mostly [Amazon pants](https://www.amazon.ca/dp/B07P3GY1ZX) and [Amazon socks](https://www.amazon.ca/dp/B07NST8SFH), but I have a variety of other clothing as well.

## Backpacks
I have a big backpack (36L) and a small backpack (18L) from IKEA; they don’t seem to sell them anymore but they’ve worked well for me over the years I’ve had them. I use the small one for day-to-day stuff, and use the big one when I’m traveling and need to bring more stuff with me.

I don’t have luggage I use when I’m traveling; I strongly prefer packing light and only taking what I can fit in my backpack.

## Hosting
I host this website with [Neocities](https://neocities.org/) which is great for hosting static sites. For my other stuff I mostly use GCP and an assortment of VPS providers.

## Development tools
I use VSCodium (open source version of VS Code without the non-free blobs) to do most of my development work. I use the [rust-analyzer](https://rust-analyzer.github.io) extension when working with Rust; it provides a great IDE experience for navigating and editing Rust codebases.

I mostly use the Chrome dev tools for reverse-engineering and analyzing web pages; it’s pretty good for debugging JavaScript. I’ve tried using the Firefox dev tools but they get really slow for me on really big JavaScript bundles.

## AI
I mostly use Claude for chatting and Claude Code for development. I’ve started using the Claude Code web interface for a few tasks recently as well (useful for making quick PRs to my personal website from my phone).

## Domains
I use [Porkbun](https://porkbun.com) for most of my domains; I use [BareMetal](https://baremetal.ca) for my .ca domains (Porkbun doesn’t support .ca).
